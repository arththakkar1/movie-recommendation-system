# Movies Dataset with Posters

The **Movies Dataset with Posters** is a curated collection of TMDB (The Movie Database) movie data, credits, and poster URLs, specifically designed for movie recommendation, data analysis, and visual projects.

## Dataset Overview

This dataset is a merged collection of three primary sources:
- **TMDB 5000 Movies dataset**: Contains movie metadata such as title, overview, genres, keywords, release date, runtime, vote count, and vote average.
- **TMDB Credits dataset**: Includes cast and crew information for each movie.
- **Poster dataset**: A curated collection of URLs for movie posters, enabling visual representation of the data.

## Key Features

The merged dataset provides a comprehensive view of each movie, including:
- **Metadata**: Overview, genres, keywords, budget, revenue, runtime, etc.
- **Credits**: Comprehensive cast and crew lists.
- **Visuals**: Curated poster URLs for every movie.

## Potential Use Cases

This dataset is ideal for:
- **Recommendation Systems**: Building content-based or collaborative filtering models.
- **Data Analysis**: Exploring trends in movie genres, cast popularity, and crew impact.
- **Visual Dashboards**: Creating interactive dashboards that display movie details alongside their posters.

## Notes & Technical Details
- **Data Quality**: All data has been cleaned and is ready for use in machine learning or visualization projects.
- **Posters**: Poster URLs were manually curated and merged. Any missing poster URLs have been filled with placeholders.
- **Source**: [Movies Dataset with Posters on Kaggle](https://www.kaggle.com/datasets/sakshisemalti/movies-dataset-with-posters)
